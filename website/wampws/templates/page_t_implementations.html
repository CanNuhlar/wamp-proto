WAMP Libraries

The following table lists WAMP 2 compliant client library implementations. For each implementation, an "x" marks a WAMP role supported by the implementation. The last column marks principle support for features from WAMP Advanced Profile.

Library Language Publisher Subscriber Caller Callee Advanced Profile
Autobahn|Python Python x x x x yes
WAMP Client & Router framework, runs under Python 2 and 3, on Twisted and Asyncio.
Autobahn|JS JavaScript x x x x yes
WAMP Client library for browsers and NodeJS using Promises (when, Q, jQuery or ES6).
Autobahn|Cpp C++ x x x x yes
C++ 11 implementation using Futures, Continuations and Lambdas. Runs on top of ASIO.
Autobahn|Android Java upcoming
WAMP Client library for native Android applications.
wampy.js JavaScript x x x x yes
WAMP Client library for browsers using callbacks. See here for comparison to Autobahn|JS.
